# BerkasPerkara

A web application deployed on GitHub Pages.

## 🌐 Live Demo
Visit: https://fadlhyfg23.github.io/BerkasPerkara/

## 📝 What is this?
BerkasPerkara is a simple web application hosted directly on GitHub Pages.

## 🚀 How to Deploy Updates
1. Make changes to `index.html` or add new files
2. Commit and push to the `main` branch
3. Changes will automatically appear at https://fadlhyfg23.github.io/BerkasPerkara/

## 📁 Project Structure
```

└── README.md       # This file
```

## 🔧 Customization
Edit `index.html` to customize your webpage. You can:
- Add CSS styles
- Add JavaScript functionality
- Create additional HTML pages
- Add CSS/JS files
